Introduction

Nécessaires

Vous avez besoin de ces plugins pour profiter pleinement de l'extension:

  1. IMountPlugin
  2. BlockParticles
  3. MmagicTitleS API
  4. PlayerAirEvents

Vous n'êtes pas obligé d'avoir chacun des addons si vous ne souhaitez pas l'utiliser, cependant, executer une effect par exemple de BlockParticles sans BlockParticles donnera une erreur.

Permets d'ajoutez des particules à vos coffres, blocs ou autre.
